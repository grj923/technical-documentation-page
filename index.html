The completed game is on Github: The Memory Game

In The Memory Game, the player clicks on any two blank white squares - hidden behind are nine pairs of images. Each turn allows two picks, if no match occurs the images are hidden once again and the player needs to remember the location of each. If a match occurs the player recieves a point. In either case, the player chooses again. With successive selections the player attempts to match images until all are matched.

The steps to build a memory game:

  • Build the HTML framework.
  • Use CSS to create the play grid.
  • Create the necessary JavaScript functions.
  • Play! Have fun! And test your memory.

createBoard Function

With the array built, you'll need to display the contents of the array on the grid. The createBoard function will do this.

Use a for loop to "loop through" the objects of the array. Each loop will assign a value to card and create an image. Test the function check the grid - it should be all blank white squares - before writing the second setAttribute and the addEventListener. 

             
              function createBoard() {
                for (let i = 0; i < cardArray.length; i++) {
                  const card = document.createElement("img");
                  card.setAttribute("src", "images/blank.png");
                  card.setAttribute("data-id", i);
                  card.addEventListener("click", flipcard);
                  grid.appendChild(card);
                }
              }

Each line of code has a purpose:

  • card.setAttribute - sets the images to blank.
  • card.setAttribute - sets each property of the object a unique "data-id" (Which will be used in a function later.)
  • card.addEventListener - listens for a click and flips a card.

flipCard Function

The flipcard function "listens" for a click of the mouse and then flips the card. Then pushes the selected card into the cardsChosen array and the cardsId pushes into the cardsChosenId array.

Steps to create the function:

  • Create a variable named cardId. Using this.getAttribute set it to the "data-id" created in the createBoard function.
  • Create an empty cardsChosen array and an cardsChosenId array. This.setAttribute will then set the src to the image chosen - flipping the card.
  • Create an if statement which checks the length of the cardsChosen array. if it reaches two, the checkForMatch function runs. If no match exists, after the timeout runs, the cards flip back. 
             
              function flipcard() {
                var cardId = this.getAttribute("data-id");
                cardsChosen.push(cardArray[cardId].name);
                cardsChosenId.push(cardId);
                this.setAttribute("src", cardArray[cardId].img);
                if (cardsChosen.length === 2) {
                  setTimeout(checkForMatch, 500);
                }
              }

checkForMatch Function

The checkForMatch function will check for three things:

  • If the player clicks the same image an alert is posted informing the player the same image was clicked.
  • If the the player makes a match the image is returned to blank, one point is added and the event listener for the two scquares is disabled.
  • If the player picks two images that do not match an alert is posted: Sorry, try again.

    Create three empty arrays:

  • cardsChosen
  • cardsChosenId
  • cardsWon

Create a variable: cards using querySelectorAll that selects all the images. Then, two const optionOneId and optionTwoId, each assigned to the first and second item in the cardsChosen array.

Create an if else statement:

  • The if statement checks if optionOneId == optionTwoId. If it is true: then, an alert is posted telling the player he has clciked the same image. The rules are that the player has to chose two diffrent sqaures to play.
  • else if: checks the two choices pushed to the cardsChosen array are an exact match. The cardsChosen arrays values were assigned with the flipcard function.

    If a match occurs:

    • An alert is posted: You have found a match.
    • Using setAttribute the images are returned to blank.
    • Then using removeEventListener the eventListener for the squares that got a match is disabled.
    • The cardsChosen is pushed into the cardsWon array.
  • If two cards that do not match are chosen: then, an alert is posted to Try Again.

             
              function checkForMatch() {
                var cards = document.querySelectorAll("img");
                const optionOneId = cardsChosenId[0];
                const optionTwoId = cardsChosenId[1];
            
                if (optionOneId == optionTwoId) {
                  cards[optionOneId].setAttribute("src", "images/blank.png");
                  cards[optionTwoId].setAttribute("src", "images/blank.png");
                  alert("You Have Clicked The Same Image");
                } else if (cardsChosen[0] === cardsChosen[1]) {
                  alert("You Have Found A Match");
                  cards[optionOneId].setAttribute("src", "images/white.png");
                  cards[optionTwoId].setAttribute("src", "images/white.png");
                  cards[optionOneId].removeEventListener("click", flipcard);
                  cards[optionTwoId].removeEventListener("click", flipcard);
                  cardsWon.push(cardsChosen);
                } else {
                  cards[optionOneId].setAttribute("src", "images/blank.png");
                  cards[optionTwoId].setAttribute("src", "images/blank.png");
                  alert("Sorry, Try Again!");
                }
