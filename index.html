INTRODUCTION

This technical page is the fourth of five projects required to earn a certificate in Responsive Web Design from FreeCodeCamp. In the beginning, I had no idea what the subject matter would be, so I designed the page utilizing placeholder text. Once I had the framework completed, I decided on a topic.

To build my JavaScript skills, Iâ€™m building a series of games using JavaScript. My technical page will be about one of those games: The Memory Game. Winning The Memory Game requires a player to click on two blank squares. After clicking on two squares, two of nine unique images are revealed. The player must remember which images were revealed and where they were on the grid. With succedding selections the player matches these images and when the player matches all images, they win.

The technical page will cover:

  • The steps to build a basic memory game using HTML, CSS and Javascript.
  • Describe the HTML used.
  • Describe the CSS used.
  • How to build a JavaScript function.
  • Describe the JavaScript functions used and how they work.
