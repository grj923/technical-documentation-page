In the beginning, I had no idea what the subject matter would be, so I designed the page utilizing placeholder text. Once I had the framework completed, I decided on a topic.

To build my JavaScript skills, Iâ€™m building a series of games using JavaScript. My technical page will be about one of the games I've completed: The Memory Game.

Winning The Memory Game requires a player to click on two blank squares. After clicking on two squares, two of nine unique images are revealed. The player must remember which images were revealed and where they were on the grid. With succedding selections the player matches these images and when the player matches all images, they win.

The steps to build a memory game:

  • Build the HTML framework.
  • Use CSS to create the game area.
  • Create JavaScript function's to allow play.
  • Play! Have fun! And test your memory.

HTML

Before adding the HTML boilerplate, create three files: index.HTML, styles.css and app.js. In the index.HTML file, (Note: I'm using VS code with the extension emmet snippets.) type HTML and select HTML 5 from the drop down menu. A standard HTML 5 biolerplate will appear.

Congratulations! You're on your way!

Link your CSS stylesheet and your JavaScript file. It's standard practice to place the JavaScript link at the bottom of the HTML file, just above the closing body tag.

The foundations of all websites - the HTML boilerplate - are typically similar. So, this is a good place to start. It will look similar to the code below. 

        
    <!DOCTYPE html>

    <html lang="en">
    
        <head>
          <meta charset="utf-8">
          <title >The Memory Game</title>
          <meta name="description" content="width=device-width, initial-scale=1.0">
          <link rel="stylesheet" href="styles.css">
        </head >

        <body> 
          <script src="js/scripts.js"></script>
        </body>

    </html>

Keep score and create the game area:

  • Inside the body tags, create a div with the id of game.
  • Inside the game div, create another div with the id of score.
  • Create a h3 heading with Score.
  • Inside the Score heading place a span with the id of result.
  • Create another div with the class of grid.

My HTML Code:

           
             <div id="game">
              <div id="score">
                <h3>
                  Score:
                  <span id="result"> </span>
                </h3>
              </div>
              <div class="grid"> </div>
            </div>
          </body>
        </html>

CSS

My goal was to practice and build upon my JavaScript skills. So, the syling was minimal.

You'll need a grid for the play area. The images I used were 100px by 100px and with twelve images, So I created a grid 300 pixels tall and 400 pixels wide.

To Create The Grid:

  • To select and style a class component such as class="grid": .grid
  • To define the flex container set display to flex
  • To wrap the containers set flex-wrap to wrap.
  • You're ready to move on to creating the functions neded to play. 
             
              .grid {
                display: flex;
                flex-wrap: wrap;
                height: 300px;
                width: 400px;
              }
